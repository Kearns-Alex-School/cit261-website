Manipulate Using JavaScript

One of the cool things that you can do with JavaScript is manipulate differnet elements on the page. This can be done with user input as buttons or text. Some of these pages have buttons that do some of this already. Here is an example of the most basic one use on this site:

// hide the button document.getElementById("b_create").style.display = "none";

As you can tell, the button is gone!

But that is not all that happened...

As you can tell, I am now visable to let you know what is going to happen next! In addition to removing the button from our sight, we now see a new text box (once again, similar to ones that you have seen on this site already). This is all possible because of JavaScript.

// remove the 'hide' class document.getElementById("step2").classList.remove("hide")

What this does is find the element with the id of 'step2' and remove the class 'hide' from its list. It does not have to be limited to a single element. Just like this step, the id is attached a div element which hides all of the children elements. Go ahead! Run the next code!
